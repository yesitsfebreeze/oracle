## Portable before proprietary

All agent must obey live in plain markdown in git, readable by anything that speak HTTP — no tool, no format, no vendor hold only copy. Adapters add convenience, never hold only copy — tool change one day, philosophy must outlive it. Behaviors propagate from upstream, not freeze into each copy: repo point at ref it trust, bump deliberate. Unpinned not propagation — that upstream editing your philosophy while you sleep.
